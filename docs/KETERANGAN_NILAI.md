## ðŸ“ Rumus Perhitungan

### 1. Nilai Mutu (NM)
```
Nilai Mutu = Bobot Ã— SKS
```

**Contoh:**
- Mata Kuliah: Pemrograman Web (3 SKS)
- Nilai Huruf: A (Bobot 4.00)
- Nilai Mutu = 4.00 Ã— 3 = 12.00

### 2. Indeks Prestasi Semester (IPS)
```
IPS = Total Nilai Mutu / Total SKS
```

**Contoh:**
```
Semester 1:
- Pemrograman Web (3 SKS, A = 4.00) â†’ NM = 12.00
- Basis Data (3 SKS, B = 3.00) â†’ NM = 9.00
- Matematika (2 SKS, A- = 3.50) â†’ NM = 7.00

Total Nilai Mutu = 12.00 + 9.00 + 7.00 = 28.00
Total SKS = 3 + 3 + 2 = 8

IPS = 28.00 / 8 = 3.50 (Sangat Memuaskan)
```

### 3. Indeks Prestasi Kumulatif (IPK)
```
IPK = Total Nilai Mutu Semua Semester / Total SKS Semua Semester
```

**Contoh:**
```
Semester 1-4:
Total Nilai Mutu = 112.00
Total SKS = 32

IPK = 112.00 / 32 = 3.50 (Sangat Memuaskan)
```

## ðŸ“ Catatan Penting

### Nilai Tidak Lulus (E)
- Nilai **E** (0.00) = **Tidak Lulus**
- Mahasiswa **wajib mengulang** mata kuliah tersebut
- SKS tetap dihitung dalam total SKS tempuh
- Nilai mutu = 0, mempengaruhi IPK

### Nilai Minimal Lulus
- Nilai minimal lulus: **C** (2.00)
- Nilai **D** dan **E** = Tidak Lulus
- Untuk mata kuliah wajib, minimal **C**
- Untuk mata kuliah pilihan, minimal **D** (tergantung kebijakan)

### Pengulangan Mata Kuliah
- Mahasiswa dapat mengulang mata kuliah dengan nilai < C
- Nilai terbaik yang diambil untuk perhitungan IPK
- Maksimal 2x pengulangan per mata kuliah

## ðŸ” FAQ (Frequently Asked Questions)

### Q: Apa bedanya IPS dan IPK?
**A:** 
- **IPS** = Indeks Prestasi Semester (hanya 1 semester)
- **IPK** = Indeks Prestasi Kumulatif (akumulasi semua semester)

### Q: Apakah nilai D lulus?
**A:** 
- Untuk mata kuliah **pilihan**: Lulus (tapi tidak direkomendasikan)
- Untuk mata kuliah **wajib**: Tidak lulus, harus mengulang

### Q: Bagaimana cara menghitung IPK?
**A:** 
```
IPK = (Total Nilai Mutu Semua Semester) / (Total SKS Semua Semester)
```

### Q: Apakah bisa mengulang mata kuliah yang sudah lulus?
**A:** 
- Ya, untuk meningkatkan nilai
- Nilai terbaik yang diambil
- Maksimal 2x pengulangan

### Q: Berapa IPK minimal untuk lulus?
**A:** 
- IPK minimal: **2.00**
- Untuk Cum Laude: **3.51**
